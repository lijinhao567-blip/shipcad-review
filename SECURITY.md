# Security Policy

## Reporting

Please report security issues privately before public disclosure.

For now, create a private communication channel with the maintainer before sharing exploit details, sensitive files, or customer data.

## Scope

Security-sensitive areas include:

- CAD file upload and parsing.
- Authentication and token handling.
- Report generation.
- Worker service boundaries.
- Model and dataset handling.

## Data Handling

Do not attach proprietary CAD drawings, credentials, production logs, or trained model files to public issues.
